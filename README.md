# PRODIGY_DS_02

## Task 02: Data Cleaning and Exploratory Data Analysis (EDA)

### Objective

Perform data cleaning and exploratory data analysis on the Titanic dataset to identify patterns, trends, and relationships between variables.

### Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

### Data Cleaning

* Filled missing values in Age using median.
* Filled missing values in Embarked using mode.
* Removed Cabin column due to excessive missing values.

### Visualizations

* Age Distribution
* Gender Distribution
* Survival Distribution
* Passenger Class Distribution
* Survival by Gender
* Survival by Passenger Class

### Key Findings

* Most passengers were between 20 and 40 years old.
* Female passengers had a higher survival rate.
* First-class passengers had better survival chances.
* Data cleaning improved dataset quality before analysis.

### Conclusion

Exploratory Data Analysis helped identify important patterns affecting passenger survival on the Titanic.
